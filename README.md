# ⏰ Django Clock App

A simple and clean digital clock web application built using Django.  
This project displays real-time time updates directly in the browser.

🌐 Live Demo:  
https://django-clock-by-gaurav.onrender.com/

---

## 📌 About

This project is a basic implementation of a real-time digital clock using Django for backend setup and HTML, CSS, JavaScript for frontend.

The main goal of this project is to understand:
- Django project structure
- Static files handling
- Real-time updates using JavaScript
- Deployment on Render

Hindi:
Ye project ek simple digital clock hai jo browser me real-time time show karta hai. Isse Django basics aur deployment samajhne me help milti hai.

---

## 🚀 Features

- Real-time clock ⏱️

## ⚙️ Installation & Setup

1. Clone the repository

```bash
git clone https://github.com/your-username/django-clock.git
cd django-clock
```


Create virtual environment
```python -m venv venv
source venv/bin/activate   # (Linux/Mac)
venv\Scripts\activate      # (Windows)
```

Run the server
```python manage.py runserver```
